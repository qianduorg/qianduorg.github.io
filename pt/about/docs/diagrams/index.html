      gráfico

      Atualmente, o Qiandu suporta o gráfico de Mermaid.

      Mermaid

      Mermaid é uma ferramenta de desenho de gráficos baseada em JavaScript. Por exemplo: fluxograma, diagrama de sequência, gráfico de pizza, etc.

      Usar o Mermaid no Hextra é tão fácil como usar blocos de código:

      ```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

      será renderizado como:

      graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
