      fórmula de látex

      $\KaTeX$ Para representar expresiones matemáticas de LaTeX. Se puede habilitar estableciendo math en true en frontmatter.

      Markdown
      ---
title: "My Page with LaTeX"
math: true
---

      Cuando esté habilitado, los scripts, hojas de estilo y fuentes de KaTeX se incluirán automáticamente en su sitio web. Esto le permite utilizar expresiones matemáticas de LaTeX en el contenido de Markdown.

      Ejemplo

      El contenido de Markdown admite expresiones matemáticas de LaTeX tanto en línea como en párrafos independientes.

      En línea

      page.md
      This $\sigma(z) = \frac{1}{1 + e^{-z}}$ is inline.

      This $\sigma(z) = \frac{1}{1 + e^{-z}}$ is inline.

      Párrafo independiente

      page.md
      $$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-j\omega t} \, dt$$

      se representará como:

      $$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-j\omega t} , dt$$

      Funciones soportadas

      Para obtener una lista de símbolos admitidos, vaya a Fórmulas admitidas por KaTeX.

      Expresión química

      Las expresiones químicas se admiten a través de mhchem.

      En línea: $\ce{H2O}$ es agua.

      Párrafo independiente:

      page.md
      $$\ce{Hg^2+ ->[I-] HgI2 ->[I-] [Hg^{II}I4]^2-}$$

      $$\ce{Hg^2+ ->[I-] HgI2 ->[I-] [Hg^{II}I4]^2-}$$
