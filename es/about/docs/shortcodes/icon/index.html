      código corto
      icono

      icono

      Para usar este código corto en línea, los códigos cortos en línea deben estar habilitados en la configuración:

      hugo.yaml
      enableInlineShortcodes: true

      Puede encontrar una lista de íconos disponibles en data/icons.yaml

      Ejemplo

      uso 

      {{< icon "github" >}}

      Heroicons iconos de esquema v1 listos para usar

      Cómo agregar tu propio ícono

      Cree un archivo data/icons.yaml y agregue sus propios íconos SVG en el siguiente formato:

      data/icons.yaml
      your-icon: <svg>your icon svg content</svg>

      Luego, esto se puede usar en un código corto como este:

      {{< icon "your-icon" >}}

{{< card icon="your-icon" >}}

      Consejo: Iconify Design es un excelente lugar para encontrar íconos SVG para su sitio web
